INTERACTION FLOW:
-----------------

1. User opens Debug Page
   ↓
2. PNG files automatically loaded from Downloads folder
   ↓
3. User taps on a PNG file in the list
   ↓
4. Selected file is highlighted
   ↓
5. Image is displayed in "Selected Image" section
   ↓
6. OCR is automatically triggered
   ↓
7. Extracted text appears in "OCR Result" section
   ↓
8. Processing details logged in Debug Output

KEY FEATURES:
-------------
- Real-time file selection and processing
- Side-by-side image and OCR result viewing
- Detailed debug logging
- Manual refresh capability
- Existing test buttons remain functional

PERMISSIONS:
------------
- READ_EXTERNAL_STORAGE (Android 12-)
- READ_MEDIA_IMAGES (Android 13+)
- Requested automatically on first access
